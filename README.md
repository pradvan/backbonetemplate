#BackboneTemplate#
Sample Backbone app that renders items in a collection and allows to add and remove items.

##Technology stack##
###Libraries###
- Backbone
- jQuery
- Bootstrap
- SASS
- jQuery.address

###Tools###
Bower for dependency management

Grunt for automation:
- JSHint
- JS & CSS minification
- JST
- JS uglify
- Image minification
- SVG minification
- HTML minification
- Concatanation
- Local Server
- Watch/Live reload

##Installation##
Make sure Node.js is installed on your system and that it is in the PATH. Clone project and run the following one at a time:

1. npm install
2. bower install
3. grunt build
4. grunt serve
