#BackboneTemplate#
Sample Backbone app that loads JSON data, renders items in a collection, allows adding and removing items.
 
##Technology stack##
###Libraries###
- Backbone
- jQuery
- Bootstrap
- SASS
- jQuery.address

###Tools###
Bower for dependency management
 
Grunt for automation:
- JSHint
- JS & CSS minification
- JST
- JS uglify
- Image minification
- SVG minification
- HTML minification
- Concatanation
- Local Server
- Watch/Live reload

##Installation##
Make sure Node.js, Rails, and Compass are installed on your system.
 
Clone project and run the following one at a time:
 
1. npm install
2. bower install
3. grunt build
4. grunt serve
